# Antithesis Quickstart

## Goal

Set up a minimal Antithesis workflow in the current project quickly and safely.

## Workflow

1. Inspect the repository layout and detect relevant runtime/build tooling.
2. Add or update minimal Antithesis configuration files needed for a first run.
3. Add a short quickstart command sequence tailored to the detected tooling.
4. Validate that added files are syntactically correct.
5. Report what was created and the exact next commands to run.

## Output

- A minimal, working first-pass Antithesis setup.
- Clear follow-up commands for local verification and iteration.
